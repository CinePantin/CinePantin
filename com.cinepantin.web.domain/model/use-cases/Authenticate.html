Scenario: authenticate
User (flow) System

In each screen, there's a login zone, with:
[loginField], [passwordField] (input fields)
[loginAction] (action)
+ a [login message] (text zone), usually empty.

The user enters data in both
[loginField], [passwordField] fields,
and click [OK]

=>

Checks both [loginField], [passwordField] against database.
Returns either:

  • SUCCESS (if combination matches)
  • FAILURE (if not)
<=
  • If SUCCESS :
    Replace login zone content with:
    [LogedInUserName] + [LogoutAction]
  • If FAILURE
    login zone with text added "Not OK, try again"
